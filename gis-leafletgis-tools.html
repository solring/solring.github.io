Leaflet

http://leafletjs.com/examples/quick-start.html 繪製GIS 可吃圖磚, GeoJSON

A simple example

<html>
<head>
  <title>Leaflet test</title>
  <link rel=stylesheet type=text/css href="http://cdn.leafletjs.com/leaflet-0.7/leaflet.css" />
    <script src="http://cdn.leafletjs.com/leaflet-0.7/leaflet.js"></script>
</head>

<body>
  <div id="map" style="height:300px; width:600px;"></div>
    <p>This is a test map</p>

<script>
var map = L.map('map').setView([25.032961, 121.562655], 14);
mapLink = 'OpenStreetMap';
        L.tileLayer('http://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', 
            {
                attribution: 'Map data &copy; ' + mapLink,
                maxZoom: 18
            }
        ).addTo(map);
</script>
</body>

</html>

Reference https://leanpub.com/leaflet-tips-and-tricks/read#leanpub-auto-html-1
